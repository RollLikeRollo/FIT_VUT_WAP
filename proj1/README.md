### How to run tests:

Tests are done using Jest.  
To install dependencies (need superuser):  
 - `sudo bash test.sh install`   

To run tests  
 - `bash test.sh`  

To uninstall dependencies  
 - `bash test.sh d`  

### How to generate documentation

Documentation generated using JDoc.    
To generate:  
 - `bash doc.sh`  

To delete  
 - `bash doc.sh d`
